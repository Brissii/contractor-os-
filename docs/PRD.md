# ContractorOS
## Product Requirements Document (PRD)

Version: 0.1

Status: Planning

---

# Vision

ContractorOS is a profit-first operating system built for landscaping and construction companies.

Unlike other software, every decision inside ContractorOS is based on profitability.

---

# Core Modules

- Dashboard
- CRM
- Leads
- Estimates
- Jobs
- Scheduling
- Employees
- Time Tracking
- Equipment
- Inventory
- Invoicing
- Reports
- AI Assistant
- Settings
